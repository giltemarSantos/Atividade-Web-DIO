Como é uma estrutura básica de um HTML?

Segue abaixo a estrutura minima para se ter um arquivo HTML

  1. DOCTYPE html >

    A tag inicial em nosso código é essencial para informar aos navegadores que o documento atual é um arquivo HTML. Essa convenção deve ser aplicada no início de todos os arquivos, garantindo sua presença constante.

  2. HTML >

    Essa tag marca o início dos elementos de uma página. Todos os outros elementos devem ser filhos diretos ou indiretos dessa tag.

  3. HEAD >

    A tag de cabeçalho (Head) abriga o conteúdo que é interpretado pela máquina, como scripts, links externos e metadados. Essa tag não representa um elemento visual.

  4. META >

    Esta tag é usada para descrever atributos adicionais de metadados inclui valores conhecidos como: charset, name...

  5. TITLE >

    Essa tag é usada para descrever o título da página exibido na guia atual do navegador.

  6. LINK >

    Essa tag estabelece uma conexão entre o arquivo atual e algum conteúdo externo, sendo mais comumente usado em conjunto com arquivos CSS. A mesma tag também pode ser utilizada para redirecionar para um ponto específico dentro da própria página web.

  7. BODY >

    É uma das tags mais populares e essenciais do HTML, onde adicionamos o conteúdo que é visualizado pelo usuário final. Essa tag é única e pode ser considerada como a tag principal de todos os elementos de conteúdo.

  8. CONTEÚDO >

    Aqui é o local onde adicionamos as tags de conteúdo, como texto, imagem, tabela e formulário. Existem muitas opções disponíveis, e é a parte do HTML em que os desenvolvedores têm maior atuação.

Entendendo a diferença entre listas ordenadas e não ordenadas

Lista não Ordenada (unordered list):

É representada pela tag ul. Nesse tipo de lista, os itens são exibidos sem uma ordem específica e são marcados por marcadores, geralmente pontos ou pequenos círculos. Os itens da lista não são numerados ou organizados em sequência. EX:

  • Item 1
  • Item 2
  • Item 3

Lista Ordenada (ordered list):

É representada pela tag ol. Nesse tipo de lista, os itens são exibidos em uma ordem sequencial e são numerados automaticamente. Os itens da lista são organizados em uma sequência específica. EX:

  1. Item 1
  2. Item 2
  3. Item 3
